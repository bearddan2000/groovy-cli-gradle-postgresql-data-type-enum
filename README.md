# groovy-cli-gradle-postgresql-data-type-enum

## Description
Creates a small database table
called `dog`. Uses an enum user defined type.

## Tech stack
- groovy
- gradle
  - 6.8.2

## Docker stack
- postgresql:alpine
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
